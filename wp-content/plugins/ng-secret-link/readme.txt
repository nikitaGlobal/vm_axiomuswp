== Installation ==

Use WordPress' Add New Plugin feature, searching "NG Secret link", or download the archive and:

1. Unzip the archive on your computer  
2. Upload plugin directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Fill in the settings in wp-admin's plugin settings mage
